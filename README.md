# ChecksumBypass
### 使用该dll你需要
* [melonloader](https://github.com/LavaGang/MelonLoader)的修改版本
* `nameTranslation.txt`（可选）
### 如何使用
* 在`Releases`中选择选择客户端对应版本的`ChecksumBypass.dll`下载
* 将`ChecksumBypass.dll`放在`melonloader`中的`mods`文件夹中
